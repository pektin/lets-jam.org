20. Templates¶

A template is similar to a type, but instead of defining all behaviour for instances, templates only describe a subset. Templates are akin to abstract base classes from other languages, but behave more like interfaces.

20.1. Syntax¶

TemplateInclude ::=  "include" Template
Template        ::=  "template" identifier
                       ClassInstructionSet
                     "end"

20.1.1. Examples¶

template Person
  name:String

  def new(n)
    name = n
  end
end

class Employee
  include Person
end

class Client
  include Person
end
