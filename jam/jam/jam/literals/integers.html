5.1. Integers¶

A integer is a whole number of the builtin type Int.

5.1.1. Syntax¶

A integer literal may contain one or more digits (0-9) with optional separation by a underscore (_) for large numbers.

5.1.1.1. Examples¶

# The speed of light (m/s)
c = 299_792_458
