6.4. Booleans¶

A boolean represents a value that can only be in two states, either true or false. Booleans are of the builtin type Bool.

6.4.1. Syntax¶

Boolean ::=  "true" | "false"

6.4.1.1. Examples¶

# Boolean variables used in a conditional block
state = true

if state == true
    print("This will print\n")
else
    print("This will not\n")
end


# Boolean value used in a conditional block
if true
    print("This will always print\n")
else
   print("This will never print\n")
end
