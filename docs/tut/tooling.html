1. Tooling¶

Jam provides a variety of tools useful for working in the language. All of these are accessible through the central jam binary.

1.1. Interactive¶

Running jam without any arguments opens up an interactive compiler session.

$ jam
Jam v0.1a interactive compiler
#|

Source can now be written directly into the terminal. Leaving a blank line will make the tool compile and run the written source. Source written between blank lines will currently not carry over.

1.2. Running¶

Invoking jam on a file compiles and runs it directly. Jam source files typically have a .jm extension.

$ jam hello_world.jm
Hello World!

Alternatively you can use the run or just r command.

$ jam run hello_world.jm
Hello World!
$ jam r hello_world.jm
Hello World!

1.3. Compiling¶

Source files can also be compiled to executable binaries using the compile or just c command.

$ jam compile hello_world.jm
$ jam c hello_world.jm
$ ./hello_world
Hello World!

Use the -o option to specify an output file name.

$ jam c hello_world.jm -o a.out
$ ./a.out
Hello World!
