5.1. Integers

A integer is a whole number of the builtin type Int.

See the builtin library for more information regarding the Integer type.

5.1.1. Syntax

Integer ::=  [0-9][ [0-9_]*[0-9] ]

5.1.1.1. Examples

# Mass of an object (kg)
mass = 10

# The speed of light (m/s)
c = 299_792_458
