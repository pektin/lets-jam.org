17.1. Constant

Constant is a modifier that enforces immutability of its value.

17.1.1. Syntax

Constant ::=  "const" ModifierValue

17.1.1.1. Example

const id = 4

id = 3 # Error! Can't be changed

name:const(String) = "Larissa"
# Same as
const name = "Larissa"
# And
name = const "Larissa"
