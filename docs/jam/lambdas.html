11. Lambdas

Lambdas are identical to methods, except that they are anonymous and therefore cannot be overloaded. They provide a quicker syntax for defining small, one use methods.

11.1. Syntax

Lambda ::=  [ "(" [ [Argument ","]* Argument ] ")" | Argument ] "=>" Value

11.2. Example

map(a => a**2, [1, 2, 3, 4]) #=> [1, 4, 9, 16]
reduce((a, b) => a*b, [1, 2, 3, 4]) #=> 24
12.times(=> puts("Hi"))
