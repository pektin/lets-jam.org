.. _jam-identifiers:

Identifiers
###########

Identifiers are used to identify values in Jam. When an identifier is used by
itself, it is understood as a reference to everything that was defined with that
same identifier.

Syntax
======

.. productionlist::
    Identifier: ([a-zA-Z_][a-zA-Z_0-9]*)

Examples
--------

::

    fooBar
    _foo_bar_fizz_bazz_
    _9
